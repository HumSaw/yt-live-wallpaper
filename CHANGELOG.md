# Changelog

Формат: [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/), версии: [SemVer](https://semver.org/lang/ru/).

## [Unreleased]

### Добавлено
- Поддержка macOS: окно-обои опускается на уровень фона рабочего стола (ниже иконок) через objc-runtime; сборка .dmg
- Поддержка Linux (X11): окно типа `desktop`; сборка .AppImage
- Кроссплатформенная загрузка yt-dlp (автономные сборки для каждой ОС)
- Release-workflow собирает пакеты для всех трёх ОС
- Интерфейс на 10 языках (русский, английский, испанский, португальский, немецкий, французский, японский, китайский, корейский, польский) с переключателем в шапке и автоопределением языка системы
- Автодокачка yt-dlp и ffmpeg при первом запуске с экраном прогресса — терминал больше не нужен
- Отмена удаления клипа (тост «Вернуть», 5 секунд)
- Кнопка «Повторить» у клипов с ошибкой загрузки
- Индикатор занятого места на диске в настройках
- Очередь загрузок: не более двух одновременных скачиваний
- ESLint, Prettier, unit-тесты (`node --test`) и CI-workflow

### Исправлено
- Обновление yt-dlp теперь работает в установленном приложении (свежий бинарник кладётся в userData вместо самообновления в Program Files)
- Распаковка ffmpeg через системный tar вместо интерполяции пути в PowerShell (падало у пользователей с апострофом в имени)
- Пути с `#` и `?` в file://-URL корректно кодируются
- Недокачанные `.part`-файлы удаляются после неудачной загрузки
- Троттлинг прогресса загрузки — интерфейс не перерисовывается сотни раз в минуту
- Обои переустанавливаются при изменении DPI/разрешения монитора на лету

### Изменено
- `main.js` сокращён до composition root (~95 строк): окна — в `window-manager.js`, IPC — в `ipc.js`, жизненный цикл — в `app-lifecycle.js`, общие флаги — в `runtime-state.js`; плейлист и трей — в отдельных модулях
- Покрытие тестами: store (персистентность, миграции, восстановление) и playlist (ротация, loop-режимы, реакция на удаление) — 24 теста
- У окна-обоев собственный минимальный preload без API панели управления
- Content-Security-Policy в обоих HTML-окнах

## [0.2.0]

### Добавлено
- Дневная тема панели с переключателем
- Паузы для экономии GPU: полноэкранные приложения, развёрнутые окна, батарея, блокировка экрана
- Поддержка нескольких мониторов (основной / выбранный / все)
- Локальные видеофайлы через drag-and-drop
- Кроссфейд между клипами
- Режимы плейлиста: один клип / друг за другом / по таймеру
- Автосборка релизов через GitHub Actions
