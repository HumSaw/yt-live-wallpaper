# YT Live Wallpaper

Живые обои для Windows: приложение само скачивает отрезки видео с YouTube (до 4K)
и проигрывает их на рабочем столе — за иконками, вместо обычной статичной картинки.

## Возможности (MVP)

- Вставляете ссылку на YouTube + таймкоды («1:20» – «1:50») — отрезок скачивается автоматически
- Качество до 4K (максимально доступное для ролика)
- Плейлист: несколько отрезков сменяют друг друга по кругу с заданным интервалом
- Пауза при полноэкранных приложениях (играх, видео) — экономия GPU
- Звук с регулятором громкости и кнопкой mute
- Автозапуск вместе с Windows
- Живёт в системном трее

## Требования

- Windows 10/11
- Node.js 20+ и pnpm (или npm)

## Установка и запуск

```bash
# 1. Установить зависимости
pnpm install

# 2. Скачать yt-dlp и ffmpeg в папку bin/ (один раз)
pnpm setup

# 3. Запустить приложение
pnpm start
```

Если используете npm: `npm install`, `npm run setup`, `npm start`.

## Как пользоваться

1. Вставьте ссылку на YouTube-видео и (по желанию) таймкоды начала и конца отрезка.
2. Нажмите «Скачать» — прогресс виден в плейлисте.
3. Как только отрезок готов, обои включаются автоматически (или нажмите «Запустить обои»).
4. Закрытие окна не завершает приложение — оно сворачивается в трей.
   Выход — через контекстное меню иконки в трее.

## Как это работает

- **Скачивание**: `yt-dlp` с флагом `--download-sections` вырезает нужный отрезок
  прямо при загрузке (точная вырезка по ключевым кадрам через ffmpeg).
- **Обои**: используется известный трюк с окном `WorkerW` — приложение просит
  Windows создать специальное окно позади иконок рабочего стола (сообщение `0x052C`
  окну `Progman`) и встраивает туда своё окно с `<video>` через `SetParent`.
- **Пауза при играх**: раз в 2 секунды проверяется, покрывает ли активное окно
  весь экран; если да — видео ставится на паузу.

## Структура проекта

```
src/
  main/
    main.js               — главный процесс: окна, трей, IPC, плейлист
    wallpaper.js          — WorkerW-трюк (koffi + user32.dll)
    downloader.js         — скачивание через yt-dlp
    fullscreen-monitor.js — детект полноэкранных приложений
    store.js              — JSON-хранилище настроек и клипов
  preload.js              — мост между процессами
  renderer/               — панель управления (UI)
  wallpaper-window/       — окно-обои с <video>
scripts/
  setup-bins.mjs          — загрузка yt-dlp и ffmpeg
bin/                      — сюда кладутся yt-dlp.exe и ffmpeg.exe
```

## Замечания

- Скачанные видео хранятся в `%APPDATA%/yt-live-wallpaper/videos`.
- 4K-видео на обоях заметно нагружает GPU — включённая пауза при полном экране
  сильно помогает в играх.
- Если YouTube начал требовать подтверждение — обновите yt-dlp: удалите
  `bin/yt-dlp.exe` и снова запустите `pnpm setup`.
