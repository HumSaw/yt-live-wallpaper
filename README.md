# YT Live Wallpaper

Живые обои для Windows: приложение само скачивает видео с YouTube (до 4K)
и проигрывает их на рабочем столе — за иконками, вместо обычной статичной картинки.

## Возможности

- Вставляете ссылку на YouTube — видео скачивается автоматически: целиком или отрезком по таймкодам
- Перетащите свой локальный видеофайл (mp4, webm, mkv, mov) прямо в окно — он попадёт в плейлист
- Качество до 4K (максимально доступное для ролика)
- Миниатюры клипов в плейлисте (кадр из видео)
- Три режима воспроизведения:
  - **Один клип** — выбранное видео крутится по кругу бесконечно
  - **Друг за другом** — видео доигрывает до конца, включается следующее, и так по кругу
  - **По таймеру** — клип сменяется каждые N секунд
- Плавный кроссфейд при смене клипов (без чёрного моргания)
- Несколько мониторов: обои на основном, на выбранном или на всех сразу
- Экономия ресурсов:
  - пауза при играх и полноэкранных приложениях
  - пауза, когда окно развёрнуто и рабочий стол не виден
  - пауза при работе от батареи (ноутбуки)
  - пауза при блокировке экрана
- Звук с регулятором громкости и кнопкой mute
- Управление из трея: запуск/стоп, следующий клип, звук
- Кнопка «Обновить yt-dlp» — если YouTube изменился и загрузки сломались
- Автозапуск вместе с Windows
- Аниме-стиль панели управления

## Требования

- Windows 10/11
- Для запуска из исходников: Node.js 20+ и pnpm (или npm)

## Запуск из исходников

```bash
# 1. Установить зависимости
pnpm install

# 2. Скачать yt-dlp и ffmpeg в папку bin/ (один раз)
pnpm setup

# 3. Запустить приложение
pnpm start
```

Если используете npm: `npm install`, `npm run setup`, `npm start`.

## Сборка установщика (.exe)

```bash
pnpm setup   # бинарники yt-dlp/ffmpeg должны лежать в bin/ до сборки
pnpm dist
```

Готовый установщик появится в папке `dist/` — например
`dist/YT Live Wallpaper Setup 0.2.0.exe`. Его можно ставить на любой ПК
с Windows — Node.js там не нужен.

## Как пользоваться

1. Вставьте ссылку на YouTube-видео. По умолчанию скачивается **всё видео целиком**;
   переключите на «Только отрезок», чтобы указать таймкоды начала и конца.
   Или просто перетащите свой видеофайл в окно.
2. Нажмите «Скачать и добавить» — прогресс виден в плейлисте.
3. Как только видео готово, обои включаются автоматически (или нажмите «Запустить обои»).
4. Выберите режим: один клип по кругу / друг за другом / по таймеру.
5. Если мониторов несколько — выберите в настройках, где показывать обои.
6. Закрытие окна не завершает приложение — оно сворачивается в трей.
   Выход — через контекстное меню иконки в трее.

## Как это работает

- **Скачивание**: `yt-dlp` с флагом `--download-sections` вырезает нужный отрезок
  прямо при загрузке (точная вырезка по ключевым кадрам через ffmpeg).
- **Обои**: используется известный трюк с окном `WorkerW` — приложение просит
  Windows создать специальное окно позади иконок рабочего стола (сообщение `0x052C`
  окну `Progman`) и встраивает туда своё окно с `<video>` через `SetParent`.
  Для нескольких мониторов создаётся по окну на каждый дисплей с пересчётом
  координат относительно виртуального экрана.
- **Кроссфейд**: в окне-обоях два слоя `<video>`; новый клип загружается в скрытый
  слой и плавно проявляется поверх старого.
- **Пауза при играх**: раз в 2 секунды проверяется, покрывает ли активное окно
  весь экран (или развёрнуто ли оно — при включённой максимальной экономии).
- **Батарея/блокировка**: события `powerMonitor` Electron.

## Структура проекта

```
src/
  main/
    main.js               — главный процесс: окна, трей, IPC, плейлист, мониторы
    wallpaper.js          — WorkerW-трюк (koffi + user32.dll)
    downloader.js         — yt-dlp: загрузка, миниатюры, обновление, ошибки
    fullscreen-monitor.js — детект полноэкранных/развёрнутых приложений
    store.js              — JSON-хранилище настроек и клипов
  preload.js              — мост между процессами
  renderer/               — панель управления (UI)
  wallpaper-window/       — окно-обои с кроссфейдом
scripts/
  setup-bins.mjs          — загрузка yt-dlp и ffmpeg
build/icon.png            — иконка приложения для сборки
bin/                      — сюда кладутся yt-dlp.exe и ffmpeg.exe
```

## Замечания

- Скачанные видео хранятся в `%APPDATA%/yt-live-wallpaper/videos`,
  миниатюры — в `%APPDATA%/yt-live-wallpaper/thumbs`.
- Локальные файлы не копируются — плейлист ссылается на оригинал.
  При удалении клипа из плейлиста ваш файл не удаляется.
- 4K-видео на обоях заметно нагружает GPU — включите паузы в настройках,
  если играете или работаете от батареи.
- Если видео перестали скачиваться — нажмите «Обновить yt-dlp» в настройках.
